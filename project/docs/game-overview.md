# Game Overview

## Concept
This project is a realtime PvP side-scrolling tower defence game where two players compete in the same live match. Each player manages offense and defense simultaneously by spawning units, placing towers, and casting abilities to overwhelm the opponent.

## Player Experience Goals
- Fast tactical decisions with constant counterplay.
- Clear feedback for every important event (spawns, kills, ability impact, objective pressure).
- Competitive fairness through authoritative server simulation.
- Strong readability even during high-action moments.

## Core Fantasy
Players feel like battlefield commanders making rapid strategic tradeoffs:
- invest mana into pressure now or save for stronger timing windows,
- reinforce defense or push offense,
- use abilities proactively or hold for reaction.

## Inspiration Pillars
- Lane pressure dynamics from tower defense.
- Realtime execution and timing mind-games from PvP arenas.
- Build diversity and adaptation from deck/loadout systems.

## High-Level Gameplay Loop
1. Enter matchmaking with selected deck/loadout.
2. Join a synchronized realtime match.
3. Spend mana on units, towers, and abilities.
4. Adapt continuously to opponent strategy and tempo.
5. Win by meeting objective victory conditions before opponent does.
6. Receive post-match rewards, ranking impact, and history updates.

## Competitive Elements
- Symmetrical starting conditions.
- Deterministic authoritative simulation.
- Server-side validation for command legality and resource use.
- Skill expression through timing, positioning, sequencing, and adaptation.

## Audience and Match Profile
- Designed for players who enjoy high-tempo strategic competition.
- Target match duration: short-to-medium sessions (approximately 6-12 minutes for MVP tuning).
- Ranked progression supported via backend profile and leaderboard systems.
