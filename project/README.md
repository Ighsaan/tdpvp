# Project Workspace

This directory contains the implementation and documentation for the realtime PvP tower defence project.

## Quick Start
Run from `/project`:

1. `make install-all`
2. `make dev-server`
3. `make dev-client`
4. Optional desktop client shell: `make dev-client-desktop`
5. Open `http://localhost:5173` (primary client)

For architecture and scope details, see:
- `/project/docs/implementation-foundation.md`
- `/project/docs/architecture-overview.md`
- `/project/docs/client-3d-rendering.md`
- `/project/docs/react-ui-shell-flow.md`
- `/project/docs/client-desktop-packaging-and-updates.md`
